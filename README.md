# Linux-windows-cpu-scaler
 Zero-dependency Linux bash script replicating Windows' native Processor Power Management (PPM) for CPU frequency scaling. Features 100ms load sampling, linear usage-to-frequency mapping, auto-detected hardware min/max/boost frequencies via sysfs. Compatible with Arch, Ubuntu, and all systemd-based distros.
